多搜索引擎手机导航页面
=========================

[项目主页](http://so.whu.link)

目前支持的搜索引擎 : )

* 百度
* Bing
* 搜狗
* 360
* 有道
* Google

环境要求
------------
* php

安装
------------
将文件直接放置在根目录即可

未来会更新的地方
----------
- 增加“Enter”发起搜索的支持 
- 增强对电脑版的友好程度
- 支持更多搜索引擎
