多搜索引擎手机导航页面
=========================

[项目主页](http://so.whu.link)

目前支持的搜索引擎 : )

* 百度
* Bing
* 搜狗
* 360
* 知乎
* 腾讯
* 金山词霸
* 有道
* Google
*

环境要求
------------
* php（不是必须的，我是需要加入php版的统计代码，不需要统计的同学直接将主页后缀改为html即可）

安装
------------
将文件直接放置在网站根目录即可
